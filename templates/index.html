Sistema Hospitalario Distribuido

Plataforma unificada para la gestión de pacientes en múltiples nodos hospitalarios

Este sistema integra las bases de datos de los hospitales de Quito y Guayaquil utilizando vistas particionadas actualizables, permitiendo operaciones CRUD transparentes sobre una base de datos distribuida.

Administra los registros de pacientes de ambos nodos hospitalarios de forma unificada.

Tecnologías: Flask, PyODBC, SQL Server, Vistas Particionadas Actualizables, Fragmentación Horizontal Primaria (FHP)
